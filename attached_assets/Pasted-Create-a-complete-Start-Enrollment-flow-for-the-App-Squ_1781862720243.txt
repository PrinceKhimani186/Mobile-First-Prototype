Create a complete "Start Enrollment" flow for the App Squad website.

When the user clicks the "Start Enrollment" button, open a dedicated Enrollment page that matches the current App Squad dark theme design, typography, spacing, glowing cyan accents, and card styles.

========================================
PAGE TITLE
========================================

Headline:
Start Your App Ownership Journey

Subheadline:
Complete your details and secure your place today.

----------------------------------------
STEP 1: USER INFORMATION
----------------------------------------

Create a clean form with these fields:

• First Name (required)
• Last Name (required)
• Email Address (required)
• Phone Number (required)

Use modern dark inputs with blue glow focus effects.

Validate:

- Email format
- Phone required
- All fields required

----------------------------------------

----------------------------------------

Show 3 pricing cards similar to the App Squad website.

Card 1

(price id here)

----------------------------------------
STEP 3: SUBMIT CONTACT TO GHL
----------------------------------------

Before payment, create/update contact in GoHighLevel.

Fields:

firstName
lastName
email
phone

If contact already exists, update it.

After successful Stripe payment, apply the corresponding tag:

Purchased Plan - Essentials

Purchased Plan - Ownership Accelerator

Purchased Plan - Digital Asset Empire

----------------------------------------
STEP 4: STRIPE CHECKOUT
----------------------------------------

Use Stripe Checkout.

Pass metadata:

firstName
lastName
email
phone
selectedPlan

After successful payment:

Trigger Stripe success webhook.

Webhook should:

1. Verify payment succeeded.
2. Find/create contact in GoHighLevel.
3. Apply the correct tag.
4. Save purchase information.

----------------------------------------
SUCCESS URL
----------------------------------------

After payment success:

Redirect user back to the main App Squad website homepage.

Example:

https://appsquad.com

Append:

?payment=success

Show success toast:

"Welcome to App Squad. Your enrollment is complete."

----------------------------------------
CANCEL URL
----------------------------------------

If payment is cancelled:

Return to enrollment page.

Show:

"Payment was cancelled. You can continue whenever you're ready."
