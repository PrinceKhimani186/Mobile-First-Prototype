==================================================
DEMO READY FOR REVIEW STAGE
==================================================

When projectStage = "Demo Ready For Review"

Display headline:
Your Demo Is Ready For Review

Display text:
Your app demo is now ready for review. Please review the design, branding, text, colors, and functionality before we move into publishing preparation.

Show 2 buttons:

Button 1:
Approve Demo

Button 2:
Submit Revision Request

==================================================
APPROVE DEMO BUTTON
==================================================

When customer clicks Approve Demo:

Show confirmation modal:

Headline:
Approve Demo?

Text:
By approving your demo, you confirm that you are ready to move forward toward publishing preparation. If you approve now, your included revision round will be marked as unused and the project will move to Final Approval.

Buttons:
Cancel
Yes, Approve Demo

When confirmed:
Set projectStage = "Final Approval"
Set demoApproved = true
Set revisionRequested = false
Set approvedAt = current timestamp

Show success message:
Your demo has been approved. Your project is now moving toward final approval and publishing preparation.

==================================================
SUBMIT REVISION REQUEST BUTTON
==================================================

When customer clicks Submit Revision Request:

Open Revision Request Form.

Form headline:
Submit Your Included Revision Request

Form helper text:
Your package includes one revision round before publishing preparation. Please include all requested changes in this submission.

Revision request form fields:

1. Revision Type
Checkboxes:
- Text / wording updates
- Color changes
- Graphic adjustments
- App icon adjustment
- Branding updates
- Minor layout changes
- Minor configuration adjustment
- Other

2. Revision Details
Large text area:
Please describe all changes requested.

3. Specific Screens / Areas
Large text area:
Which screens, images, buttons, colors, or sections need changes?

4. Priority Level
Dropdown:
- Low
- Normal
- High

5. Upload Reference Image
File upload placeholder:
Optional reference image or example

6. Confirmation Checkbox
Text:
I understand this is my included revision round and major feature additions, new game mechanics, new templates, or out-of-scope changes may require additional fees.

Submit Button:
Submit Revision Request

==================================================
AFTER REVISION REQUEST SUBMISSION

When revision form is submitted:

  package: ""

Show success message:
Your revision request has been submitted. Our team will review your notes and update your project status once revisions are complete.
